# K3 Weekly AI Brief — Quick Reference

## Title Format
`Keel3_AI_Brief_[Client]_W[N]_[Year]`

## Assembly Steps (at a glance)
1. **Pull Stories** — fetch candidates from the AI Brief Story Pool
2. **Filter** — narrow to stories matching the target client + current week
3. **Select 4–5** — choose stories most relevant through the client's lens
4. **Tailor** — rewrite each story in client-specific context and tone
5. **Render PDF** — generate the one-page brief and save to File Hub

## Key Database
- **AI Brief Story Pool** (Notion)
- Filter by: client name, Candidate status, source date within target week

## Reminders
- Always verify the date window before selecting stories
- One brief per client per week — no duplicates
- Save final PDF to the project's File Hub
