# BUILD_PLAN_v1

## What I'm Building
A polished 4-page personal portfolio website for a fictional product designer (Patrick Reyes), deployed as static HTML.

## How It Will Work
- Four standalone HTML pages sharing one CSS file
- No framework or build step — clean, fast static HTML + CSS + minimal JS
- Fixed navigation bar linking all pages with active-state highlighting
- Dark theme with purple accent, card-based layouts, and responsive design

## What I'll Create
- [x] `code/index.html` — Landing page with hero section, stats, and brand strip
- [x] `code/about.html` — Bio, 6-card skills grid, career timeline
- [x] `code/projects.html` — 4 project cards with filter buttons and CTA
- [x] `code/contact.html` — Contact info, social links, and a functional form with success state
- [x] `code/styles.css` — Shared stylesheet (nav, typography, buttons, cards, footer)
- [x] Live preview deployed
- [x] Saved to project

## Questions / Assumptions
- Used realistic placeholder content (fictional designer persona)
- No backend — form shows a JS success animation without sending emails
