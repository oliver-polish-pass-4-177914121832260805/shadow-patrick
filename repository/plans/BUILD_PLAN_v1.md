# Build Plan: Habit Tracker

## What I'm Building
A simple single-page habit tracker with a clean, minimal interface.

## How It Will Work
- Display a title at the top
- Input field to add new habits
- List of habits with checkboxes to mark them complete
- Data persists in browser storage so habits stay between visits
